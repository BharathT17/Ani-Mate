# Ani-Mate
Ani-Mate Is A Website Which Is Developed On The Concept Of <b> Dynamic Database Content Driven using HTML,CSS and PHP.</b><br>
Users Can <b>Login</b> With Their Email Address And Password and The Password Will Be Stored In The Encrypted Form. Then Follow The Prompts To <b>View Shows or Movies</b> According To
Their Interest.They Can Give <b>Feedback</b> on Website.<br>
Admin Can <b>Add Movies Or Series</b> Using DataForm Page And The New Added Item Will Be Shown In The Home Page.
